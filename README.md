# Gelo Digital Studio

A premium, dark-tech agency site for a full-stack developer offering websites, web apps, and mobile apps.

## Stack

- Vite + React + TypeScript
- Tailwind CSS

## Scripts

- npm run dev
- npm run build
- npm run preview

## Notes

- The contact form is wired to Formspree in App.tsx.
- Replace the Facebook button link when you are ready.
